# Abilash — Portfolio

Personal portfolio site built with React 19 + Vite 8.

## Stack

- **React 19** — UI framework
- **Vite 8** — build tool
- **Pure CSS** — dark/light theme via CSS variables
## Development
```bash
npm install
npm run dev
```
## Build
```bash
npm run build
```
The production output goes to `dist/`.
## Deploy
Push to GitHub; Render auto-deploys from the `main` branch. A `public/_headers` file is included for correct MIME types on Render's CDN.
